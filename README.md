溶液配制计算器


一个功能完整的化学溶液配制计算工具。支持固体称量、母液稀释、多种单位转换、溶解度检查和可打印的配制步骤指南。

## 核心功能

### 计算模式
- **从固体配制**：输入目标浓度与体积，自动计算所需称量质量
- **由母液稀释**：使用 C1V1=C2V2 原理计算所需取液体积

### 单位支持

**浓度单位**：M (mol/L)、mM (mmol/L)、μM (μmol/L)、g/L、mg/L、μg/L、% w/v、ppm、ppb

**体积单位**：L、mL

### 溶质数据库
- 内置常用溶质库（NaCl、KCl、葡萄糖、NaOH、Tris、CaCl₂）
- 包含摩尔质量、溶解度、水合物信息
- 数据持久化到浏览器 `localStorage`
- 支持导入/导出 JSON 或纯文本（CSV/TSV）
- 可编辑、添加、删除条目
- 恢复默认数据库选项

### 配制步骤指南
- 自动生成详细的操作步骤
- 包括所需仪器、准备工作、具体操作、检查与保存
- 区分固体称量和母液稀释两种模式
- **打印友好**：可直接打印或导出为 PDF

### 其他功能
- 摩尔质量自动计算（支持水合物）
- 溶解度检查与警告提示
- 实时消息反馈
- 清晰的两标签页 UI（计算器 | 数据库管理）

## 快速开始

1. **打开页面**：在浏览器中打开 `index.html`

2. **在计算器标签页中输入**：
   - 溶质名称（可从下拉列表选择）
   - 摩尔质量（自动填充或手动修改）
   - 水合物数（如果是水合物，如 1 表示一水合物）
   - 目标浓度和单位
   - 目标体积和单位
   - 选择模式（固体配制或母液稀释）

3. **点击计算**，查看结果和配制步骤

4. **打印/导出**：点击"打印/导出 PDF"按钮保存步骤指南

## 数据库管理

在**溶质数据库**标签页中：

- **导入**：上传 JSON 文件或纯文本（CSV/TSV）
- **添加**：逐条输入溶质信息
- **编辑/删除**：修改或删除表格中的条目
- **导出**：保存为 JSON 文件
- **重置**：恢复默认数据库

### 纯文本格式
```
名称, 摩尔质量(g/mol), 溶解度(g/L), 水合物数(可选)
NaCl, 58.44, 359, 0
KCl, 74.55, 344
```

## 使用示例

**配制 100 mL 0.1 M NaCl 溶液**：
1. 选择 NaCl
2. 浓度：0.1 M，体积：100 mL
3. 模式：从固体配制
4. 点击计算
5. 结果：需称量 0.5844 g NaCl，自动生成操作步骤

## 重要说明

- ✅ 所有计算都在浏览器本地进行，无需联网
- ✅ 数据保存在浏览器 `localStorage` 中
- ⚠️ 清除浏览器数据会丢失自定义数据库条目
- ⚠️ 溶解度检查仅在数据库有记录时生效
- ⚠️ 配制步骤是通用参考，实际操作需根据实验要求调整
- ℹ️ 建议定期导出数据库备份
